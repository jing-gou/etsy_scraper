# 🛒 Etsy 高清图片采集工具使用指南

这是一款专为 Etsy 设计的高清商品图片自动采集工具。它能够根据关键词搜索商品，并深入详情页抓取该商品所有的展示原图，自动分类保存。
## 🛠️ 运行前准备

1. **代理软件**：由于 Etsy 的网络限制，运行前请务必开启代理工具（如 Clash、V2Ray 等）。
* 请确保代理模式为“全局”或“规则模式”。
* 记住你的本地代理端口（默认通常是 `10808` 或 `7890`）。


2. **浏览器**：本工具优先调用电脑自带的 **Google Chrome** 浏览器，如果没有，则尝试调用 **Microsoft Edge**。


## 🚀 使用步骤

1. **启动程序**：
双击运行 `Etsy高清图片采集器.exe`。
2. **配置参数**：
* **搜索关键词**：输入你想抓取的商品名称（建议用英文）。
* **爬取页数**：建议先输入 `1` 进行测试。
* **代理端口**：输入你代理软件的实际端口（如果不确定，请检查代理软件设置）。


3. **开始运行**：
点击“开始运行”按钮，程序会自动弹出一个受控制的浏览器窗口。
4. **重要：人工辅助（如有必要）**：
* 如果浏览器弹出 **"Unusual activity"（检测到异常流量）** 的验证页面，请**手动**在浏览器窗口中点击验证按钮。
* 验证通过进入搜索结果页后，程序会自动接管并开始下载。**采集期间请勿关闭该浏览器窗口**。



## 📂 文件保存路径

* 程序会在 `.exe` 同级目录下创建一个以**关键词**命名的文件夹。
* 每个商品会根据其 `Listing ID` 创建独立的子文件夹。
* 图片均以最高清原图（`fullxfull`）格式保存。



## ⚠️ 注意事项

* **不要频繁操作**：短时间内大量抓取可能会导致 IP 被 Etsy 临时封锁，建议分批次运行。
* **下载失败**：如果日志显示下载失败，通常是代理网络不稳定，请尝试更换代理节点。
* **防病毒软件**：由于程序经过加密打包，部分杀毒软件可能会误报，请选择“信任”或“允许运行”。



## 🔧 故障排除

| 现象 | 原因 | 解决方法 |
| --- | --- | --- |
| 浏览器显示 403 错误 | 代理失效或节点被封 | 切换代理节点并重启程序 |
| 找不到详情页链接 | Etsy 页面结构更新 | 联系开发者更新 CSS 选择器 |
| 程序启动后秒退 | 端口被占用或环境冲突 | 检查 10808 端口是否被其他软件占用 |
